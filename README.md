# Bank-Loan-Analysis - Tableau Project

It is a comprehensive Bank Loan Analysis project using Tableau that provides deep insights into lending operations and portfolio performance. This project demonstrates end-to-end data analytics, visualization, and business intelligence skills. 📊
 
PROJECT HIGHLIGHTS:
 
📌 3 Interactive Dashboards:
 
1️⃣ SUMMARY DASHBOARD - Key Performance Indicators & Business Metrics
   • Total Loan Applications, Funded Amount & Received Amount tracking
   • Month-to-Date (MTD) & Month-over-Month (MoM) analysis
   • Average Interest Rate monitoring
   • Debt-to-Income Ratio (DTI) assessment
   • Good Loan vs Bad Loan Analysis (distinguishing based on loan status)
   • Comprehensive Loan Status Grid View
 
2️⃣ OVERVIEW DASHBOARD - Trend Analysis & Pattern Recognition
   • Monthly Trends by Issue Date (Line Charts) - identifying seasonality and long-term lending patterns
   • Regional Analysis by State (Filled Maps) - geographical performance insights
   • Loan Term Distribution (Donut Charts) - understanding term preferences
   • Employee Length Impact (Bar Charts) - employment history correlation
   • Loan Purpose Breakdown (Bar Charts) - primary borrowing reasons
   • Home Ownership Analysis (Tree Maps) - hierarchical home status impact
 
3️⃣ DETAILS DASHBOARD - Comprehensive Data Portal
   • Consolidated view of all essential loan data
   • Single source for detailed portfolio insights
   • User-friendly interface for critical information access
